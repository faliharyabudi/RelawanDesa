# RelawanDesa - Sistem Manajemen Kegiatan Sosial dan Relawan

RelawanDesa adalah sistem fullstack dengan tema Smart Village yang memfasilitasi kegiatan sosial dan pendaftaran relawan. Proyek ini memadukan backend NestJS, admin dashboard Next.js, dan aplikasi mobile React Native untuk relawan.

## Teknologi Utama
- **Backend:** NestJS (TypeScript), MySQL, Prisma ORM, JWT Auth
- **Frontend (Web Admin):** Next.js (React), TailwindCSS
- **Mobile (User):** React Native (Expo)

## Struktur Proyek
- `backend/` - REST API backend
- `frontend-admin/` - Web CMS untuk admin
- `mobile-user/` - Aplikasi mobile untuk relawan
